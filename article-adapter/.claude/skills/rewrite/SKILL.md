# Skill: Рерайт статьи

Полная переделка текста по стилевой системе — без контекста конкретного издания. Использует промт `articles/promts/mode-rewrite.md`.

## Триггеры вызова

- «Перепиши [статью]»
- «Сделай рерайт»
- «Переделай статью»
- «Усиль текст»
- «Переделай этот черновик целиком»

## Когда использовать вместо адаптации

| | Адаптация | Рерайт |
|---|---|---|
| Указано конкретное издание | да | нет |
| Нужен ресёрч сайта издания | да | нет |
| Подтягиваются параметры из CRM | да | нет |

## Краткий алгоритм

1. **Диагностика (5-7 предложений)**: где текст теряет читателя
2. **Переписать** по 8 принципам письма
3. **Выдать в формате**:
   - Диагностика
   - Новый заголовок + 3 альтернативы
   - Новая система подзаголовков
   - Улучшенная версия статьи
   - Что изменено и почему
   - Что проверить вручную

## Ограничения по умолчанию

- Объём 4000–7000 знаков (если Соня не указала иначе)
- Стиль — деловое медиа
- Без ИИ-пафоса, канцелярита, рекламной подачи
- Брендовый блок + анонс — обязательно
