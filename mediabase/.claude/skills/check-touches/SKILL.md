# Проверка касаний

Локальный аналог напоминаний (Соня выбрала вариант «по запросу», не cron). Запускается, когда Соня сама спрашивает.

## Триггеры

- «Проверь касания»
- «Кому пора писать»
- «Напоминалка по СМИ»
- «Что с касаниями»
- «Кого надо подергать»

## Логика интервалов (согласована с Соней)

- **2-е касание** — через 7 дней после 1-го, если нет ответа
- **3-е касание** — через 14 дней после 2-го, если нет ответа
- После 3-х касаний без ответа — статус автоматически в **холодный** (после подтверждения)

Эти числа заложены в `sheets.log_touch(..., next_touch_days=N)`. По умолчанию 7. Pitch при отправке 2-го касания вызывает с `next_touch_days=14` для 3-го напоминания.

## Алгоритм

### Шаг 1. Найти просроченные

```python
import sheets
overdue = sheets.find_overdue_touches()  # все где следующее_касание <= сегодня и статус ≠ отказ
```

Также можно показать те, у которых `следующее_касание` в ближайшие 3 дня (превентивно).

### Шаг 2. Сгруппировать

```
ПРОСРОЧЕНО (нужно писать сейчас):
  • [издание] — [тема последнего касания], просрочено на N дней, статус [...]
  • ...

СЕГОДНЯ/ЗАВТРА:
  • [издание] — [тема], следующее касание [дата]
```

### Шаг 3. Для каждого предложить действие

```
Что делаем с [издание]?
(а) Записать новое касание (отправил тему X сегодня)
(б) Перенести напоминание на +N дней
(в) Поставить статус «отказ» (3 касания без ответа)
(г) Пропустить
```

После выбора Сони:
- (а) → `sheets.log_touch(site, today, тема, '', next_touch_days=N)` где N=7 для 2-го, =14 для 3-го
- (б) → `sheets.update_publication(site, следующее_касание=new_date)`
- (в) → `sheets.update_publication(site, статус_прогрева='отказ', следующее_касание='')`

## Что обязательно прочитать

- `mediabase/sheets.py` — функции `find_overdue_touches`, `log_touch`, `update_publication`
- `pr-helper/shared/zerocoder-context.md` — для контекста о бренде при формулировке темы нового касания

## Что НЕ делать

- Не обновлять статусы без подтверждения
- Не предполагать, что отсутствие ответа = «отказ» — спросить Соню
- Не игнорировать поле `комментарий` — там могут быть пометки вроде «не писать до сентября»
